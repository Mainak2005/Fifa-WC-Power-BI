🌍 FIFA World Cup Dashboard (1930–2014)

📊 Project Overview
This project presents an interactive Power BI dashboard analyzing historical FIFA World Cup data from 1930 to 2014. The dashboard provides insights into tournament trends, country performances, hosting patterns, and match statistics.

The goal of this project is to transform raw football data into meaningful visual insights that can help understand the evolution of the World Cup over time.

🚀 Key Features
🌎 Country & Attendance Analysis
Visual map showcasing geographical distribution of attendance
Highlights global engagement and popularity of the tournament
🏆 World Cup Overview
Central visual representing the tournament timeline (1930–2014)
Acts as a thematic anchor for the dashboard
📈 Competition Stats by Year

Tracks:
Goals Scored
Matches Played
Qualified Teams
Helps identify trends like:
Increase in teams over time
Growth in total goals and matches

🏟️ Hosting Nations Analysis
Bar chart showing countries that have hosted the World Cup
Identifies frequently hosting nations like:
Brazil
France
Germany
Italy

🥇 World Cup Winners Distribution
Donut chart displaying:
Number of titles per country
Highlights dominant teams such as:
Brazil
Italy
Germany

📉 Winners Trend Analysis
Line chart showing number of wins by country
Provides a comparative view of football dominance over time

🛠️ Tools & Technologies
Power BI – Data visualization & dashboard creation

📂 Dataset
Historical FIFA World Cup dataset (1930–2014)
Includes:
Match statistics
Goals scored
Participating teams
Host countries
Winners

📌 Insights Derived
Brazil is the most successful team historically
Expansion of teams led to increased match count and goals
European and South American countries dominate World Cup wins
Hosting is concentrated among a few nations
